# 扫雷游戏

## 项目介绍
该项目是一个简单的扫雷游戏，玩家可以选择不同的难度级别进行游戏。游戏的目标是找到所有非雷区而不触发地雷。

## 文件结构
```
扫雷
├── src
│   ├── main.c        // 程序的入口点，包含主函数，负责显示菜单并处理用户输入
│   ├── game.c        // 实现游戏的主要逻辑，包括初始化棋盘、设置地雷、处理玩家移动等
│   └── game.h        // 声明游戏逻辑中使用的函数和常量
├── Makefile          // 用于构建项目，定义编译规则和目标
└── README.md         // 项目的文档，介绍游戏的玩法和编译运行说明
```

## 游戏玩法
1. 启动游戏后，玩家可以选择入门级别或进阶级别。
2. 玩家通过输入坐标来选择要揭开的格子。
3. 如果玩家揭开了地雷，游戏结束。
4. 如果玩家成功揭开所有非雷区，游戏胜利。

## 编译和运行
1. 确保你的系统上安装了C编译器（如gcc）。
2. 在项目根目录下打开终端。
3. 运行以下命令以编译项目：
   ```
   make
   ```
4. 编译完成后，运行游戏：
   ```
   ./扫雷
   ```
